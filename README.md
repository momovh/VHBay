# VHBay


Dự án VHBay:
- Phát triển trên: Node.js
- Framework: Express.js (Back-end), Preact (Front-end)
- Version: 1.0.0
- Cơ sở dữ liệu: MongoDB
- HTTP Library: Axios
- Quản lý trạng thái: Redux
